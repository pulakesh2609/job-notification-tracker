# Job Notification Tracker

Welcome to the **Job Notification Tracker** repository by KodNest!

## Overview
Job Notification Tracker is a web application designed to help users track, save, and digest job notifications with confidence. It features a clean, responsive interface built with modern web technologies.

## Features
- **Dashboard**: Get an overview of all your job notifications.
- **Saved**: Keep track of the job notifications you care about the most.
- **Digest**: Digest job notifications efficiently.
- **Settings**: Customize your tracking experience.
- **Responsive Design**: Works seamlessly across desktop and mobile devices.

## Tech Stack
- **HTML5**
- **CSS3** (with Custom Properties for theming)
- **JavaScript (ES6+)**
- **Tailwind CSS** (for rapid UI refinement)

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/pulakesh2609/job-notification-tracker.git
   ```
2. Navigate to the project directory:
   ```bash
   cd job-notification-tracker
   ```
3. Open `index.html` in your favorite web browser to view the application locally.
